In a successful test, 

1) the csproj file will have lines 7,8 and 9 blanked out
2) the .csproj.vspscc file will be deleted
3) the vssver.scc file will be deleted
4) a folder named scr_log will be created 
5) a file named scr_log.txt will be created in the scr_log folder, containing a list of all actions performed.


To run the application
Copy the EXE into the project folder in question and run.
